Grocery chains face $18B in shrink annually with outdated markdown practices. Expired Solutions built a proprietary GPT + Computer Vision model and prototype for real-time produce freshness scoring and markdown automation.

## Features

- **Real-time Freshness Scoring**: Computer vision model for produce assessment
- **Automated Markdown**: Dynamic pricing based on freshness
- **Retailer & Shopper Apps**: iOS apps for both sides
- **QR Code System**: For easy in-store integration

## Tech Stack

- **Swift (iOS)**, **GPT-4**, **Vision AI**, **Azure**, **Firebase**, **QR Code System**

## Impact

- Up to **20% shrink reduction** projected in pilot
- Pitched to Giant Eagle; McGinnis Venture Competition finalist
- Shopper and retailer-facing apps built and demoed
